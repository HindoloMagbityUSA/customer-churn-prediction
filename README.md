# Customer Churn Prediction

An end-to-end machine learning project to predict customer churn using the CRISP-DM framework.

# Problem Statement
Customer churn is a major business risk. This project aims to identify customers likely to churn so that proactive retention strategies can be applied.

# Framework
CRISP-DM:
- Business Understanding
- Data Understanding
- Data Preparation
- Modeling
- Evaluation
- Deployment (conceptual)

# Tech Stack

- pandas, NumPy, scikit-learn
- Jupyter Notebook
